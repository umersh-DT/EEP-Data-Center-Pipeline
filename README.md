Historically, EEP faced steep institutional financial pressures, recording severe multi-billion Birr comprehensive net losses between FY 2021/22 and FY 2022/23. This project demonstrates how the calculated integration of high-density data centers transformed the utilityвЂ™s operational model. By absorbing vast, under-monetized baseload power (such as energy generated by the Grand Ethiopian Renaissance Dam - GERD) at near-flat maximum continuous load factors, these facilities became crucial anchor consumers. 

Crucially, this pipeline tracks and separates the **dual-benefit economic engine** generated by this rollout:
1. **Utility Utility Inflows (EEP):** Direct energy invoicing paid in foreign currency ($220M+ USD), helping fund infrastructure modernization, grid repair, and rural electrification.
2. **Sovereign Fiscal Revenue (Customs Commission):** Upfront import tariff fees and handling duties collected at border clearance hubs (like Kality) by the **Ethiopian Customs Commission (ECC)**, flowing straight into the national treasury to benefit the broader federal budget.

## рџ› пёЏ Data Pipeline Architecture & Sources

The project repository is structured across three clean pipeline layers:

* **`data/raw/eep_baseline.csv`**: Grounded in official institutional performance tracking data, incorporating key performance parameters including:
  * **HS Commodity Codes 8471 & 8504** data structures tracking digital processing units (high-performance servers) and industrial static power converters.
  * **Capital Goods Incentives Frameworks**: Adjusting models to account for customs duty exemptions under national ICT investment regulations, tracking baseline processing tariffs.
* **`data/processed/eep_ecc_integrated_metrics.csv`**: Formulates multi-agency data joins and engineers predictive infrastructure leading indicators.
* **Predictive 90-Day Deployment Lag**: Features a built-in shift variable mapping the physical latency gap between hardware border entry clearance and active grid megawatt draw.
